## Descripción

Proyecto Final del curso de backend NestJS, UPEA.

## Clonar el repositorio

```bash
$ https://github.com/M4uSpY/Proyecto-CursoNestJS-Plataforma-Musical.git
```

## Instalar las dependencias

```bash
$ npm install o npm i
```

## Subir el contenedor a Docker

```bash
$ docker compose up -d
```

Verificar en el Docker Desktop que este corriendo el contendor.

## Crear el archivo de variables de entorno

```bash
.env
```

Copiar el contenido del archivo .env.example

## Corrida del Backend

```bash
npm run start:dev
```

## Swagger

Para acceder a la documentación de los endpoints creados, dirigirse a la siguiente URL

```bash
http://localhost:3000/docs
```
